# JVM 监控工具

- jvisualvm
  - 功能：
    - 能监控 CPU、类、堆、元空间、线程等基本信息
    - 支持远程
    - 支持快照读入和堆dump出
    - 可执行手动垃圾回收
- JConsole
  - 功能
    - 能监控堆、线程、类、内存等基本信息
    - 可执行手动GC
    - 可查询启动基本信息

- jmap和jstat

- jps和jcmd

- jstack、jmc 和 jhat
